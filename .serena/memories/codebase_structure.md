# Codebase Structure

- `src/song_shake/`: Source code package
    - `main.py`: Entry point (Typer app)
    - `auth.py`: Authentication logic
    - `enrichment.py`: Gemini integration and enrichment logic
    - `playlist.py`: YouTube Music playlist interactions
    - `storage.py`: TinyDB storage operations
- `pyproject.toml`: Project configuration and dependencies
- `README.md`: Documentation
- `songs.db`: Local database (TinyDB)
