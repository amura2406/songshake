# Song Shake

Song Shake is a CLI tool that supercharges your YouTube Music playlists by enriching them with granular genre and mood metadata using Google's powerful Gemini 3 Flash model.

## Features
- Secure authentication with YouTube Music.
- Automatic audio download using `yt-dlp`.
- IP enrichment using Gemini 3 Flash Preview.
- Persistent storage in `songs.db` (TinyDB).
- Cost tracking for Gemini API usage.
